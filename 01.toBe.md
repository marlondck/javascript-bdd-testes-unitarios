- Realiza a comparação com o operador `====`, que compara o valor e seu tipo.

```javascript
describe("Suite de testes toBe", () => {
  let v1 = true;
  let v1Copia = v1;
  let v1Texto = "true";

  it("deve validar o uso de matcher 'toBe'", () => {
    expect(v1).toBe(true);
    expect(v1Copia).toBe(v1);
    expect(v1Texto).toBe("true");
  });
});
```
